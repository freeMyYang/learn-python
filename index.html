2.6 变量与常量

变量和常量是编程的基础概念，用于存储和表示数据。

区别：

变量：变量可以被看作是存储数据的容器。在Python中，变量不需要显式声明其类型。当你给变量 赋值时，Python会自动确定变量的类型。

变量的命名规则：

  • 变量名必须以字母（a-z，A-Z）或下划线（_）开头。
  • 变量名的其他部分可以由字母、数字（0-9）或下划线组成。
  • 变量名是大小写敏感的，例如，age和Age是两个不同的变量。
  • 不能使用Python中的关键字作为变量名。

常量：常量是指其值一旦被设置就不能改变的变量。虽然Python本身没有内置对常量的直接支持， 但是按照惯例，我们会使用全大写字母（例如'PI'）来表示一个变量应该被视为常量，以此向 其他程序员表明这个变量的值不应该被改变。

示范：使用变量和常量计算圆的面积
xxxxxxxxxx
PI = 3.14#定义常量‘PI’存储圆周率
r = input("请输入圆的半径: ")#定义变量‘r’存储用户输入的数据


#将输入的字符串转换为浮点数
#浮点数介绍：在Python中，浮点数是用来表示实数的数据类型，它可以表示小数点后有数字的数值
r = float(r)


#输出面积
print("圆的面积是:", PI*r*r)

2.7 使用海龟画图

1.初始化和创建
  • import turtle或from turtle import *: 导入Turtle模块。
  • turtle.Screen(): 创建一个窗口以供海龟画图。
  • turtle.Turtle(): 创建一个新的海龟对象，用于绘图。
2. 移动命令
  • turtle.forward(distance): 向前移动指定的距离。
  • turtle.backward(distance): 向后移动指定的距离。
  • turtle.right(angle): 向右转动指定的角度。
  • turtle.left(angle): 向左转动指定的角度。
  • turtle.goto(x, y): 移动到指定的坐标位置。
  • turtle.setx(x): 将海龟移动到指定的x坐标。
  • turtle.sety(y): 将海龟移动到指定的y坐标。
  • turtle.circle(radius): 绘制一个圆，参数为半径。
3. 画笔控制命令
  • turtle.pendown() 或 turtle.pd(): 放下画笔，开始绘制。
  • turtle.penup() 或 turtle.pu(): 抬起画笔，停止绘制。
  • turtle.pensize(width): 设置画笔的宽度。
  • turtle.pencolor(color): 设置画笔的颜色，颜色可以是字符串如"red"或RGB值。
  • turtle.fillcolor(color): 设置填充颜色。
4. 形状和速度
  • turtle.speed(speed): 设置海龟的移动速度，范围[0,10]。
5. 填充命令
  • turtle.begin_fill(): 开始填充图形。
  • turtle.end_fill(): 结束填充图形。
6. 其他命令
  • turtle.clear(): 清除窗口，不移动海龟。
  • turtle.reset(): 清除窗口，并将海龟恢复到初始状态。
  • turtle.hideturtle(): 隐藏海龟的图标。
  • turtle.showturtle(): 显示海龟的图标。

3.1 算数运算符

  1. 加法运算符（+）：用于计算两个数值的和。

    xxxxxxxxxx
    result = 3 + 5  # 结果为 8

  2. 减法运算符（-）：用于计算两个数值的差。

    xxxxxxxxxx
    result = 10 - 2  # 结果为 8

  3. 乘法运算符（*）：用于计算两个数值的乘积。

    xxxxxxxxxx
    result = 4 * 2  # 结果为 8

  4. 除法运算符（/）：用于计算两个数值的商，结果总是浮点数。

    xxxxxxxxxx
    result = 16 / 2  # 结果为 8.0

  5. 整除运算符（//）：用于计算两个数值的商，并将结果向下取整到最接近的整数。

    xxxxxxxxxx
    result = 17 // 2  # 结果为 8

  6. 取余运算符（%）：用于计算两个数值相除的余数。

    xxxxxxxxxx
    result = 18 % 10  # 结果为 8

  7. 幂运算符（**）：用于计算一个数值的另一个数值次幂。

    xxxxxxxxxx
    result = 2 ** 3  # 结果为 8

  1. abs(x)：返回一个数的绝对值。

    示例：

    xxxxxxxxxx
    num = -10

第四课：用分支结构控制计算

4.1 单分支if

单分支if语句用于在满足某个条件时执行特定的代码块。如果条件为真，则执行if后面的代码块，否则跳过该代码块。

xxxxxxxxxx
age = input("你的年龄是：")
age = int(age)  # 将输入的年龄转换为整数类型


if age >= 18:
    print("你已经成年了！")

4.2 双分支if

双分支if语句用于在满足某个条件时执行特定的代码块，同时在条件不成立时执行另外一段代码块。如果条件为真，则执行if后面的代码块，否则执行else后面的代码块。

xxxxxxxxxx
age = input("你的年龄是：")
age = int(age)  # 将输入的年龄转换为整数类型


if age >= 18:
    print("你已经成年了！")
else:
    print("你还未成年！")

5.3 循环嵌套

循环嵌套是指在一个循环内部包含另一个循环，这种结构常用于处理多维数据、矩阵遍历等场景。在 Python 中，您可以方便地使用 for 循环或 while 循环进行嵌套。

下面是一个简单的示例，展示了如何使用循环嵌套：

xxxxxxxxxx
# 使用嵌套循环打印 i 和 j 的组合
for i in range(3):  # 外层循环，i 的取值范围为 0、1、2
    for j in range(2):  # 内层循环，j 的取值范围为 0、1
        print(f"i: {i}, j: {j}")  # 打印 i 和 j 的当前取值组合

输出结果：

xxxxxxxxxx
i: 0, j: 0
i: 0, j: 1
i: 1, j: 0
i: 1, j: 1
i: 2, j: 0
i: 2, j: 1

在这个示例中，外层的 for 循环迭代变量 i 的值为 0、1、2，而内层的 for 循环迭代变量 j 的值为 0、1。因此，内层的循环体会在外层每次循环时都执行。

类似地，您也可以嵌套 while 循环，只需在内部循环中添加另一个 while 循环即可。

需要注意的是，循环嵌套可能会导致代码逻辑复杂，因此在使用时应谨慎考虑，以避免使代码难以理解和维护。

5.4 循环控制

  1. break：终止循环。当程序执行到 break 语句时，循环会立即结束，程序将跳出循环体执行后续的代码。
  2. continue：跳过当前迭代。当程序执行到 continue 语句时，循环会立即进入下一次迭代，跳过本次迭代内 continue 之后的代码。
  3. pass：占位符。pass 语句通常用作占位符，表示暂时不做任何操作，保持代码结构完整。

示例：

xxxxxxxxxx
for i in range(5):
    if i == 2:
        print("Skipping 2")
        continue  # 跳过当前迭代
    if i == 4:
        print("Breaking at 4")
        break  # 终止循环
    print(i)
else:
    print("Loop completed")

输出结果：

xxxxxxxxxx
0
1
Skipping 2
3
Breaking at 4

在这个示例中，当 i 等于 2 时，使用 continue 跳过了打印数字 2 的步骤；当 i 等于 4 时，使用 break 终止了整个循环。同时，else 分支中的代码没有被执行，因为循环被 break 终止了
